## 🎨 设计理念

### 核心设计原则
> **Shaker的设计哲学**：简约而不简单，专业而不疏远，让每个用户都能轻松找到完美的那一杯。

- **🍸 专业优雅**：体现鸡尾酒文化的精致和品味
- **🎯 简洁直观**：去除不必要的装饰，专注核心功能
- **📱 响应灵活**：完美适配各种设备和屏幕尺寸
- **✨ 愉悦体验**：流畅的动画和贴心的交互细节
- **🌙 氛围营造**：用色彩和光影营造调酒的浪漫氛围

## ✨ 动画和过渡

### 1. 动画时长
```css
/* 动画时长标准 */
--duration-fast: 0.1s;      /* 快速 - 按钮反馈 */
--duration-normal: 0.2s;    /* 标准 - 悬停效果 */
--duration-slow: 0.3s;      /* 慢速 - 页面切换 */
--duration-slower: 0.5s;    /* 更慢 - 复杂动画 */
```

### 2. 缓动函数
```css
/* 缓动函数 */
--ease-out: cubic-bezier(0, 0, 0.2, 1);          /* 自然减速 */
--ease-in: cubic-bezier(0.4, 0, 1, 1);           /* 自然加速 */
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);     /* 平滑过渡 */
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55); /* 弹性效果 */
```

### 3. 常用动画类
```css
/* 淡入动画 */
.fade-in {
  animation: fadeIn var(--duration-normal) var(--ease-out);
}

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* 滑入动画 */
.slide-up {
  animation: slideUp var(--duration-normal) var(--ease-out);
}

@keyframes slideUp {
  from { transform: translateY(20px); opacity: 0; }
  to { transform: translateY(0); opacity: 1; }
}

/* 缩放动画 */
.scale-in {
  animation: scaleIn var(--duration-normal) var(--ease-bounce);
}

@keyframes scaleIn {
  from { transform: scale(0.9); opacity: 0; }
  to { transform: scale(1); opacity: 1; }
}
```

## 🎨 品牌应用指南

### 1. Logo使用规范
- **最小尺寸**：24px高度（数字环境）
- **安全距离**：Logo高度的1/2
- **色彩变体**：全彩版、单色版、反白版
- **禁用规则**：不得拉伸变形、不得添加效果

### 2. 色彩应用
- **主色调**：用于CTA按钮、链接、重要信息
- **辅助色**：用于文本、边框、背景
- **功能色**：用于状态提示、警告信息

### 3. 图标风格
- **风格**：线性图标，2px描边
- **尺寸**：16px、24px、32px标准尺寸
- **色彩**：继承文本颜色或使用主色调
